Mention All For Messenger

If you want to use this extension on m.me or www.messenger.com link, make sure you're logged in with the same account that you use on www.facebook.com.

1. Navigate to either of 2 links below:
  • https://www.facebook.com/messages/
  • m.me
2. Open the group chat which you want to mention all

3. Right-click on any position to open context menu

4. Click "Mention all"
